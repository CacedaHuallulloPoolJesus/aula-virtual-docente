# ISO/IEC 27001 - Seguridad de la Información

## 1. Introducción

ISO/IEC 27001 establece un Sistema de Gestión de Seguridad de la Información (SGSI).

---

## 2. Objetivo

Proteger la información académica almacenada por el Sistema Integral de Aula Virtual Docente.

---

## 3. Controles implementados

- Login institucional.
- Roles de usuario.
- Protección mediante NextAuth.
- Variables de entorno (.env).
- Base de datos protegida.
- Control de sesiones.
- Restricción de acceso a rutas.

---

## 4. Evidencias

| Evidencia | Ruta |
|------------|------|
| Login | src/app/login |
| NextAuth | src/app/api/auth |
| Variables | .env |
| Prisma | prisma/schema.prisma |

---

## 5. Beneficios

- Protección de información.
- Integridad de datos.
- Disponibilidad del sistema.
- Control de acceso.
