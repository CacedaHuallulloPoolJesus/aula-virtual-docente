# Normativas ISO aplicadas

Este proyecto considera las siguientes normas internacionales:

- ISO 9001:2015
- ISO/IEC 25000 (SQuaRE)
- ISO/IEC 29119
- ISO/IEC 27001

Estas normas sustentan la gestión de calidad, calidad del software, pruebas y seguridad del Sistema Integral de Aula Virtual Docente.
