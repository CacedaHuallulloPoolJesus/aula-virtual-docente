# ISO 9001:2015 - Sistema de Gestión de la Calidad

## Sistema Integral de Aula Virtual Docente

## 1. Introducción

La norma ISO 9001:2015 establece los requisitos para implementar un Sistema de Gestión de la Calidad (SGC), promoviendo la mejora continua, la satisfacción del usuario y el control de los procesos.

En el presente proyecto se adopta esta norma como referencia para organizar el desarrollo del software, documentar los procesos y controlar las versiones del producto.

---

## 2. Objetivo

Aplicar principios de gestión de calidad durante el desarrollo del Sistema Integral de Aula Virtual Docente.

---

## 3. Principios aplicados

- Enfoque al usuario.
- Liderazgo.
- Participación del equipo.
- Enfoque basado en procesos.
- Mejora continua.
- Toma de decisiones basada en evidencia.
- Gestión documental.
