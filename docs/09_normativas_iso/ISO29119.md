# ISO/IEC 29119 - Pruebas de Software

## 1. Introducción

La norma ISO/IEC 29119 establece buenas prácticas para planificar, ejecutar y documentar pruebas de software.

---

## 2. Objetivo

Garantizar que el Sistema Integral de Aula Virtual Docente funcione correctamente antes de su despliegue.

---

## 3. Tipos de pruebas realizadas

- Pruebas funcionales.
- Pruebas de autenticación.
- Pruebas de navegación.
- Pruebas de base de datos.
- Pruebas de generación de sesiones mediante IA.
