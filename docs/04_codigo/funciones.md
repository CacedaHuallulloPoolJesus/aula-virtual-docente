# Funciones Principales del Sistema

| Función / Archivo | Módulo | Descripción |
|-------------------|--------|-------------|
| auth.ts | Autenticación | Configura la autenticación y control de sesión. |
| prisma.ts | Base de datos | Inicializa la conexión con Prisma ORM. |
| api-client.ts | Servicios | Centraliza peticiones HTTP del frontend. |
| validations.ts | Validaciones | Contiene reglas de validación para formularios. |
| ai-session-generator.ts | IA Sesiones | Genera sesiones de aprendizaje a partir de datos ingresados por el docente. |
| agents.ts | Agentes IA | Define agentes pedagógicos simulados o configurados. |
| pdf-export.ts | Reportes | Genera archivos PDF para exportación. |
| pdf-server.ts | Reportes | Apoya la generación de documentos desde el servidor. |
| mock-data.ts | Demo | Contiene datos de demostración para uso local. |
