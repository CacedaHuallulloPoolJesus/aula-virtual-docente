# Requerimientos del Sistema

## Objetivo

Describir los requerimientos funcionales y no funcionales del Sistema Integral de Aula Virtual Docente.

---

# Requerimientos Funcionales

RF01 Iniciar sesión.

RF02 Registrar docentes.

RF03 Registrar estudiantes.

RF04 Registrar grados y secciones.

RF05 Registrar asistencia.

RF06 Registrar calificaciones.

RF07 Gestionar sesiones.

RF08 Generar sesiones mediante IA.

RF09 Consultar dashboard.

RF10 Generar reportes.

RF11 Exportar PDF.

RF12 Configurar parámetros del sistema.

---

# Requerimientos No Funcionales

RNF01 Seguridad mediante autenticación.

RNF02 Base de datos PostgreSQL.

RNF03 Diseño responsive.

RNF04 Tiempo de respuesta menor a 3 segundos.

RNF05 Disponibilidad del sistema.

RNF06 Compatibilidad con navegadores modernos.
